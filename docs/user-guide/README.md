# Guía de Usuario

## Registro y Autenticación

### Registro

1. Acceder a la página de registro
2. Completar el formulario con:
   - Nombre completo
   - Email
   - Contraseña
3. Verificar email
4. Iniciar sesión

### Inicio de Sesión

1. Ingresar email y contraseña
2. Opción de "Recordarme"
3. Recuperación de contraseña
